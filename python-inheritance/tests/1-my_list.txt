=======================================
Testing the MyList class
=======================================

First, we need to import the class:
>>> MyList = __import__('1-my_list').MyList

Check instantiation:
>>> my_list = MyList()
>>> type(my_list) == MyList
True

Check inherits from list:
>>> issubclass(MyList, list)
True

Check __str__:
>>> my_list = MyList([1, 2, 3])
>>> print(my_list)
[1, 2, 3]

Check append():
>>> my_list = MyList()
>>> my_list.append(1)
>>> print(my_list)
[1]

Check print_sorted() with sorted append:
>>> my_list = MyList()
>>> my_list.append(1)
>>> my_list.append(2)
>>> my_list.print_sorted()
[1, 2]

Check print_sorted() with not sorted append:
>>> my_list = MyList()
>>> my_list.append(5)
>>> my_list.append(2)
>>> my_list.print_sorted()
[2, 5]

Check print_sorted() with no sorted append with negative number:
>>> my_list = MyList()
>>> my_list.append(4)
>>> my_list.append(-3)
>>> my_list.print_sorted()
[-3, 4]

Check print_sorted() with empty list:
>>> my_list = MyList()
>>> my_list.print_sorted()
[]

Check print_sorted() returns a new list (doesn't mutate the original):
>>> my_list = MyList()
>>> my_list.append(10)
>>> my_list.append(2)
>>> my_list.print_sorted()
[2, 10]
>>> print(my_list)
[10, 2]
